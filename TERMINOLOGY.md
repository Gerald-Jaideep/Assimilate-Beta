# Terminology Standardization

To ensure consistency across the Case Presentation Manager, the following clinical and professional terms have been standardized:

| Old Term | Standardized Term | Context |
|----------|-------------------|---------|
| Clinical Intelligence Forge | Case Presentation / Submission Wizard | General naming of the platform/process |
| Forge AI Clinical Intelligence | Generate Thumbnail | AI-driven image generation |
| Video Intelligence Stream URL | Google Meet/Zoom Link | Live session links |
| Deploy Intelligence | Submit Case | Final submission action |
| Intelligence Focus | Case Title | Primary title of a case |
| Deployment Status | Submission Status | State of the case (Draft/Scheduled/Published) |
| Clinical Intelligence Narrative | Case Description | Detailed text of the case |
| Neural Visual | Clinical Illustration / Thumbnail | AI generated visuals |
| Partner Onboarding | Sponsor Management | Collaboration with industry |
| Intellectual Barrier | Access Restriction | Paywall or auth requirement |

## Consistency Guidelines
1. **Clinical Tone**: Use "Present" instead of "Deploy".
2. **Medical Professionalism**: Use "Expert" or "Specialist" instead of "Genius" (if any).
3. **Accuracy**: Always include ICD-10 codes for specializations.
